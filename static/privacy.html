Confidentialité

⚠️ Vous utilisez la version en ligne.
Dans cette version, vos fichiers transitent par notre serveur hébergé sur Render.com, puis sont supprimés automatiquement. Pour un traitement 100% privé, téléchargez l'application locale.

Vos fichiers restent sur votre machine

FileLabs fonctionne entièrement en local. Lorsque vous traitez un fichier, celui-ci est envoyé uniquement à un serveur qui tourne sur votre propre ordinateur (127.0.0.1). Il ne transite jamais par internet, ni par nos serveurs, ni par des services tiers.

Aucune collecte de données

Nous ne collectons aucune donnée personnelle. Il n'y a ni cookies de tracking, ni analytics, ni publicité. Aucun identifiant utilisateur n'est créé ou stocké.

Suppression automatique des fichiers

Les fichiers traités sont stockés temporairement dans un dossier local (outputs/) sur votre machine, puis supprimés automatiquement au bout d'une heure. Vous pouvez aussi les supprimer manuellement via le bouton "Nouveau" après chaque traitement.

Bibliothèques utilisées

Le traitement des fichiers utilise uniquement des bibliothèques open source (Pillow, pikepdf, FFmpeg, pdf.js) qui s'exécutent localement. Ces bibliothèques ne transmettent aucune donnée à des serveurs externes.

Code source

FileLabs est open source. Vous pouvez inspecter l'intégralité du code sur GitHub.
